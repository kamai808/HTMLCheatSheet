HTML Cheat Sheet

Document Outline
<!DOCTYPE html>Type of document
<html>HTML document
<head>Document metadata
<body>Page contents
Lists
<ol>Ordered list
<ul>Unordered list
<li>List item
Document Structure
<div>Container to group content
<h[1-6]>Headings
<span>Inline container
<p>Paragraphs
Tables
<table>Creates a table
<thead>Table header
<tbody>Table body
<tr>Creates a row
<th>Heading for either row or column
<td>Table cell
